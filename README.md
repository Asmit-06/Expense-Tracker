# Expense Tracker

A full-stack Expense Tracker application built using React, Tailwind CSS, Node.js, Express.js, and MongoDB.

The application helps users track income and expenses, manage transactions, balances, and gain insights into their spending habits.

# Features

## Dashboard

- View Total Balance
- View Total Income
- View Total Expenses
- View Total Transactions
- Monthly Income Comparison
- Monthly Expense Comparison
- Monthly Balance Comparison
- Monthly Transaction Comparison

## Transaction Management

- Add Transactions
- Edit Transactions
- Delete Transactions
- Search Transactions
- View Recent Transactions
- Dedicated Transaction Page
